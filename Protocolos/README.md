# Viaje completo de una petición web

## Descripción

Este proyecto consiste en el análisis del funcionamiento de una comunicación web desde que un usuario introduce una URL en el navegador hasta que la página es mostrada en pantalla.

El objetivo es comprender cómo interactúan los principales protocolos de red utilizados durante este proceso y cuál es la función de cada uno de ellos.

## Contenido

Se estudian los siguientes protocolos y tecnologías:

- ARP
- DHCP
- DNS
- NAT
- TCP/IP
- HTTPS

Además, se explica:

- Qué ocurre desde que se escribe una URL hasta que se carga una página web.
- Qué dispositivos participan durante la comunicación.
- Qué protocolos intervienen en cada fase.
- Qué consecuencias tendría el fallo de cada uno de ellos.

## Objetivos de aprendizaje

- Comprender el funcionamiento de una comunicación en red.
- Relacionar los distintos protocolos de Internet.
- Entender el papel de routers, servidores DNS y servidores web.
- Analizar el recorrido completo de una petición HTTP/HTTPS.

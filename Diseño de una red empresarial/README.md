# Diseño de una red empresarial

## Descripción

Este proyecto consiste en el diseño de una infraestructura de red para una empresa de aproximadamente 100 empleados.

El objetivo es proponer una red segura, organizada y escalable mediante la segmentación por VLANs, un plan de direccionamiento IP y reglas básicas de comunicación entre los diferentes departamentos y servicios de la empresa.

## Objetivos

- Diseñar una red empresarial funcional.
- Aplicar segmentación mediante VLANs.
- Definir un esquema de direccionamiento IP.
- Proteger la información mediante reglas de acceso entre redes.
- Justificar las decisiones tomadas durante el diseño.

## Escenario

La empresa cuenta con:

- 100 empleados.
- Departamento financiero.
- Departamento técnico.
- Red para invitados.
- Servidores internos.
- Acceso para teletrabajadores mediante VPN.

## Contenido del proyecto

- Diagrama de la red.

- Reglas básicas de segmentación.
- Justificación técnica de la infraestructura propuesta.

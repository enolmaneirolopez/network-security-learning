# Portfolio de proyectos - Redes y Ciberseguridad

## Descripción

Este repositorio contiene una recopilación de proyectos y prácticas relacionadas con redes, sistemas y ciberseguridad.

Los proyectos están orientados a comprender el funcionamiento de las comunicaciones, analizar infraestructuras de red y aplicar conocimientos técnicos mediante documentación, análisis y diseño de soluciones.

## Proyectos incluidos

## 📡 Protocolos de red: funcionamiento de una comunicación web

Proyecto centrado en comprender qué ocurre realmente cuando un usuario accede a una página web.

Se analiza el recorrido completo de una petición desde el equipo cliente hasta el servidor, estudiando cómo interactúan los diferentes protocolos implicados.

Protocolos analizados:

- ARP.
- DHCP.
- DNS.
- NAT.
- TCP/IP.
- HTTPS.

Conceptos trabajados:

- Resolución de nombres mediante DNS.
- Asignación de configuración de red mediante DHCP.
- Comunicación entre dispositivos mediante TCP/IP.
- Traducción de direcciones mediante NAT.
- Establecimiento de conexiones seguras mediante HTTPS.

---

## 🔎 Análisis de tráfico con Wireshark

Proyecto enfocado en la observación y análisis del tráfico real de una red utilizando Wireshark.

Se estudian diferentes comunicaciones para comprender qué paquetes se intercambian y qué información puede obtenerse mediante el análisis del tráfico.

Capturas realizadas:

- Navegación web.
- Resolución DNS.
- Renovación DHCP.
- Conexión SSH.

Conceptos trabajados:

- Análisis de paquetes.
- Flujo de comunicación cliente-servidor.
- Protocolos de red.
- Información visible durante una captura.
- Importancia del cifrado en las comunicaciones.

---

## 🌐 Diseño de red empresarial

Proyecto dedicado al diseño de una infraestructura de red para un entorno empresarial.

Se plantea una solución de red teniendo en cuenta aspectos de organización, conectividad, seguridad y escalabilidad.

Conceptos trabajados:

- Diseño de topologías de red.
- Direccionamiento IP.
- Segmentación de redes.
- Dispositivos de red.
- Seguridad de infraestructura.
- Organización de una red empresarial.

---

## 🛠️ Proyecto integrador

Proyecto final donde se combinan los conocimientos adquiridos durante los diferentes trabajos realizados.

Consiste en la planificación y desarrollo de una solución técnica completa aplicando conceptos de redes, sistemas y seguridad informática.

Conceptos trabajados:

- Análisis de requisitos.
- Diseño de soluciones.
- Implementación técnica.
- Documentación profesional.
- Resolución de problemas.

---

## Tecnologías y herramientas utilizadas

- Wireshark.
- TCP/IP.
- DNS.
- DHCP.
- HTTPS.
- SSH.
- Herramientas de diseño y documentación de redes.

## Objetivos

- Comprender el funcionamiento de las comunicaciones en red.
- Analizar protocolos y tráfico real.
- Diseñar infraestructuras de red funcionales y seguras.
- Mejorar la capacidad de documentación técnica.
- Aplicar conocimientos de redes y ciberseguridad en proyectos prácticos.
